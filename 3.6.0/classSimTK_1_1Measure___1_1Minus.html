Detailed Description

template<class T>
class SimTK::Measure_< T >::Minus

This Measure is the difference of two Measures of the same type T.

Template Parameters
TAny type that supports a subtract operator that returns the difference as another object of type T. In particular, Real, Vec<N>, and Vector will work.
